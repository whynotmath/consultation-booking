# 학생·학부모 상담 신청 Streamlit 앱

2026년 7월 27일부터 8월 7일까지 평일 상담을 신청받는 앱입니다. 상담 시간은 12:00부터 16:00까지 한 시간 단위이며, 같은 시간의 중복 신청을 막습니다.

## 신청 항목

- 학번과 학생 이름
- 신청자: 학생 본인, 부, 모, 부·모 모두
- 상담 방식: 전화 상담, 대면 상담
- 상담 날짜와 시간
- 정해진 시간이 어려운 경우 기타 희망 일정 요청
- 연락처와 상담 희망 내용

연락처는 `010-0000-0000` 형식으로만 접수됩니다.

학생 이름은 제출 시 비공개 `학생명렬` 시트와 대조되며, 일치하는 학번이 관리 시트에 자동 기록됩니다. 전체 학생 이름과 학번은 공개 앱에 노출되지 않습니다.

학부모 신청 화면에는 전화 상담 권장 안내가 표시됩니다.

## Apps Script 배포

1. `학생·학부모 상담 신청 관리` Google Sheet를 엽니다.
2. `확장 프로그램 → Apps Script`를 선택합니다.
3. 기본 편집기의 내용을 지우고 저장소의 `Code.gs` 내용을 붙여넣습니다.
4. 첫 줄의 `여기에_관리_시트_ID를_입력하세요`를 실제 관리 시트 ID로 바꾼 후 저장합니다.
5. `배포 → 새 배포 → 유형 선택 → 웹 앱`을 선택합니다.
6. 실행 사용자는 `나`, 액세스 권한은 `모든 사용자`로 설정합니다.
7. 권한을 승인한 후 `/exec`로 끝나는 웹 앱 URL을 복사합니다.

## Streamlit 배포

1. Streamlit Community Cloud에서 이 저장소의 `app.py`를 선택합니다.
2. Advanced settings의 Secrets에 아래처럼 입력합니다.

```toml
apps_script_url = "복사한_웹_앱_URL"
```

3. Deploy를 누릅니다.

Apps Script URL 외의 Google 인증키는 필요하지 않습니다.
